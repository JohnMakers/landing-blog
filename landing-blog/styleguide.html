How to customize

  1. Open assets/styles.css.
  2. Under :root change --color-primary or --color-accent to new hex values.
  3. Save. If Live Server is running, the page reloads so you can see the change.
  4. To adjust spacing or radius, edit --space-* or --radius.
  5. For dark theme tweaks, modify the variables inside [data-theme="dark"].
